# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

🔗 **Live Demo:** https://rinki049.github.io/Tic-Tac-Toe-Game-Project/


📂 **Repository:** https://github.com/Rinki049/Tic-Tac-Toe-Game-Project



## Features

*  Two-player gameplay
*  Clean and responsive UI
*  Sound effects for moves and winning
*  Restart game functionality
*  Winner detection logic



##  Technologies Used

* HTML5
* CSS3
* JavaScript 



## 📁 Project Structure

```
├── index.html
├── style.css
├── script.js
├── game music.mp3
├── ting.mp3
├── win.mp3
├── exited_2.gif
```



##  How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/Rinki049/Tic-Tac-Toe-Game-Project.git
   ```
2. Open the folder
3. Run `index.html` in your browser
